# yarn_sub_calculator
This yarn substitution calculator calculates how many balls of a yarn you'll need if you are substituting yarn for your project. It lets you know if you're getting close to the end of your last ball and recommends having an extra. 

It's a Python tool that can demonstrate resource buffers and modulo logic to middle school students. If the math shows that users are within 25% of running out, the code automatically recommends a 'safety' unit. 

This could help students connect classroom math (remainders) to professional inventory and gaming logic. The could even use it to plan enough slices of pizza are ordered for a party.
